# Lokahi Lessons

Interactive, student-facing learning resources built on the **Meaning-First Literacy™** framework by [Lokahi Connect](https://lokahiconnect.org) — a 501(c)(3) nonprofit advancing equitable literacy for neurodivergent learners.

---

## Two kinds of lessons

This repository holds two distinct types of resources:

**`/program`** — Lessons developed for Lokahi Connect's instructional program and public digital library. Grounded in Structured Word Inquiry (SWI) and INT-informed design. Intended for broad use by students, families, and educators.

**`/student`** — Lessons developed for individual students during direct services. Built around a specific learner's needs, interests, or goals. Not designed for general use, though some may be adapted for the program library over time.

## Accessibility features

All program lessons include or are being built toward:

- **🔊 Text-to-speech** — student-initiated read-aloud with a natural voice; never auto-plays
- **✋ Drag-to-answer** — drag-and-drop word bank as an alternative to typing
- Sufficient contrast, font size ≥ 14pt, and no flashing animations throughout

---

## Developmental bands (Meaning-First Literacy™)

| Band | Grades | Focus |
|------|--------|-------|
| Band 1 — Emergent | K–1 | Oral-to-print connection, early word family noticing |
| Band 2 — Early Structural | 2–3 | Word sums, mini-matrices, join conventions |
| Band 3 — Transition to Generative | 3–4+ | Full matrices, etymology, academic vocabulary bridges |
